
# Zenith Board - Task Management Application

Zenith Board is a high-performance, responsive Kanban board application built with **React**, **TypeScript**, and **Tailwind CSS**. It allows users to manage tasks across different stages with a clean, professional interface.

##  Features

- **Drag and Drop Interface:** Smoothly move tasks between 'To Do', 'In Progress', and 'Done' columns.
- **Persistent Storage:** Uses LocalStorage to ensure your tasks are saved even after page refresh.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Task Management:** Easy to add, edit, and delete tasks with real-time UI updates.
- **Modern UI:** Built using Shadcn UI components for a premium look and feel.

##  Tech Stack

- **Frontend:** React.js (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand / React Hooks
- **Components:** Shadcn UI / Lucide React
